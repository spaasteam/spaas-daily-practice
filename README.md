### 今日题目 将数组中的所有元素相乘

```js
制作一个 `multiplyAll` 函数，该函数接受一个整数数组，然后返回一个函数，然后接受一个倍数参数，最后返回一个原数组与倍数相乘后的新的数组。

example
multiplyAll([1, 2, 3])(2) = [2, 4, 6];

```
